## Installation

Get code using

```
git clone https://github.com/syahrulafrizal/bions-challange-code.git
```

__Step 2__

Install the necessary libraries (this proyect use Flutter 3.22.3 & Dart >=3.4.4 <4.0.0)

```
flutter pub get
```

__Step 3__

Run the app

```bash
flutter run
```
